User Documentation

Documentation of the phc API, including how to write plugins, has been moved to the Developer Manual.

phc has limited support for generating code, and can be used as a front-end to parse PHP for other applications.

The Usage documentation below explains how to compile and run phc, and how to convert from PHP to XML and back.

Note that you can download the documentation for this and for older versions from the archive. Moreover, although we have tried to document phc as well as we can, if anything is still unclear, please let us know by sending an email to the mailing list.
